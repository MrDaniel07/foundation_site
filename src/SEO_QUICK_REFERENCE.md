## After Deployment

### 1. Install Dependencies & Build:
```bash
npm install
npm run build
```

### 2. Deploy to Netlify:
```bash
netlify deploy --prod --dir=dist
```

### 3. Submit to Google:
1. Go to https://search.google.com/search-console
2. Add your site
3. Submit sitemap
